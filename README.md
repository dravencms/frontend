# DravenCMS base

This is a Draven CMS base

This package currently handles these things:
* Base Component
* Base Form Component
* Base Grid Component
* Routing Interface
* Templating Interface
* Base Presenter

## Instalation

The best way to install dravencms/base is using  [Composer](http://getcomposer.org/):


```sh
$ composer require dravencms/base
```
